Umbraked - SDL2 Game

---------------------------------------------
🎮 Giới thiệu:
Umbraked là một trò chơi được phát triển bằng C++ và thư viện SDL2. 
Bạn sẽ điều khiển nhân vật chính vượt qua các thử thách hấp dẫn.

---------------------------------------------
🛠️ Công nghệ sử dụng:
- Ngôn ngữ: C++
- Thư viện đồ họa: SDL2
- IDE: Code::Blocks

---------------------------------------------
🕹️ Hướng dẫn chơi:
- Phím A/D: Di chuyển nhân vật
- Phím W: Nhảy
- Phím SPACE: Bắn 
- Tránh chướng ngại vật và đat điểm số cao nhất.

---------------------------------------------
⚙️ Cài đặt:
1. Tải file nén (zip) về máy.
2. Giải nén ra thư mục bất kỳ.
3. Click đúp vào file 'Umbraked.exe' để chơi game.

*Lưu ý:* Nếu thiếu file '.dll', hãy chắc chắn bạn đã cài SDL2 đầy đủ hoặc để kèm các file DLL trong thư mục game.
